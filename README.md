# Aplikasi Hitung Volume Bangun Ruang (Kivy)

## Deskripsi
Aplikasi ini dibuat menggunakan **Python Kivy** untuk menghitung volume bangun ruang:
- Kubus
- Tabung
- Bola

Pengguna bisa memasukkan nilai sisi/jari-jari/tinggi, lalu menekan tombol `Hitung` untuk melihat hasil.

---

## Fitur
- Hitung volume Kubus
- Hitung volume Tabung
- Hitung volume Bola
- Validasi input: menampilkan popup error jika input tidak valid
