# Aplikasi Hitung Volume Bangun Ruang

## Deskripsi
Aplikasi ini dibuat menggunakan Python Tkinter untuk menghitung:
- Volume Kubus
- Volume Tabung
- Volume Bola

## Fitur
- Menghitung Volume Kubus
- Menghitung Volume Tabung
- Menghitung Volume Bola
- Tampilan GUI sederhana

## Cara Menjalankan
1. Install Python
2. Download project ini
3. Jalankan file main.py
4. Masukkan nilai lalu klik tombol hitung

## Rumus yang Digunakan
Kubus  : s³  
Tabung : π r² t  
Bola   : 4/3 π r³
